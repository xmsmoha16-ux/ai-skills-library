# مكتبة مهارات الوكيل الذكي (AI Skills Library)

مستودع منظم لتغذية أي AI Agent بمهارات حقيقية قابلة للاستخدام مباشرة.

## الأقسام
- [`programming/`](./programming) — خوارزميات، تصميم أنظمة، أنماط برمجية
- [`engineering/`](./engineering) — أفضل ممارسات هندسة البرمجيات
- [`languages/`](./languages) — مرجع سريع للغات البرمجة
- [`education/`](./education) — أطر ومنهجيات التعلم الفعّال

كل ملف مصمم ليكون قابلاً للتغذية المباشرة لأي Agent (عبر Notion, Pinecone, أو قراءة مباشرة من GitHub) كمصدر معرفة موثوق.

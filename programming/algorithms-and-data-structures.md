# خوارزميات وهياكل بيانات أساسية

## هياكل البيانات الأساسية
- **Array / List**: وصول O(1)، إدراج/حذف في النهاية O(1)، في الوسط O(n)
- **Hash Map**: بحث/إدراج/حذف متوسط O(1)
- **Linked List**: إدراج/حذف في البداية O(1)، بحث O(n)
- **Stack (LIFO)**: مفيد في undo/redo، تقييم التعبيرات
- **Queue (FIFO)**: مفيد في جدولة المهام، BFS
- **Binary Search Tree**: بحث/إدراج/حذف O(log n) لو متوازنة
- **Heap**: أسرع طريقة للوصول لأصغر/أكبر عنصر، O(log n) إدراج/حذف
- **Graph**: لتمثيل العلاقات (شبكات، خرائط، تبعيات)

## خوارزميات البحث والفرز الأساسية
- **Binary Search**: O(log n) على مصفوفة مرتبة
- **Merge Sort / Quick Sort**: O(n log n) متوسط
- **BFS**: أقصر مسار في رسم بياني غير موزون
- **DFS**: استكشاف المسارات والدورات
- **Dijkstra**: أقصر مسار في رسم بياني موزون (أوزان موجبة)
- **Dynamic Programming**: حل المشاكل بتقسيمها لمشاكل فرعية متداخلة مع تخزين النتائج (memoization)

## قاعدة ذهبية لاختيار الأداة
اختر البنية حسب العملية الأكثر تكرارًا (قراءة، كتابة، بحث) وليس حسب الأسهل تنفيذًا.
